# Git ที่ควรเรียนควบคู่ไปเลย

ระหว่างสร้างเว็บไซต์นี้ ให้ใช้ Git ตั้งแต่วันแรก

```bash
git init
```

สร้าง Commit ทุกครั้งที่เสร็จงาน

```bash
git add .
git commit -m "chapter 01 complete"
```

สร้าง Branch เวลาลองฟีเจอร์ใหม่

```bash
git checkout -b dark-mode
```

Merge กลับ

```bash
git merge dark-mode
```

คุณจะได้เรียน Git จากการใช้งานจริง ไม่ใช่จากทฤษฎี
