# Git Mastery

เว็บไซต์สอน Git ระดับ Professional — vanilla HTML/CSS/JS ไม่มี framework ไม่มี build step

## รัน

เปิด `index.html` ด้วย browser ตรง ๆ ได้เลย หรือใช้ live server:

```bash
# ตัวเลือก 1: เปิดไฟล์ตรง ๆ
start index.html        # Windows

# ตัวเลือก 2: live server (reload อัตโนมัติ)
npx serve .             # ถ้ามี Node
python -m http.server   # ถ้ามี Python
```
